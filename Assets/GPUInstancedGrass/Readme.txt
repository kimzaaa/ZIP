Thankyou for using our grass asset!

To set up, easily dragged in the prefab "Grass Example" to your scene to look
at how to set it up.

If you want to change grass settings, go to the Terrain gameobject.
If you want to change the camera settings, go to the Main Camera.
If you want to change the FPS Counter, go to the Game Manager.

For more information, check the video in the assetstore to set it up from scratch.
